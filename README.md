# Clustering_analysis_segment_potential_high_value_clients
Although the dataset does not explicitly include tax-deductible expenditures or asset portfolios, proxy variables 
such as PURCHASES and INSTALLMENTS_PURCHASES were used to represent spending behavior. 
CREDIT_LIMIT and BALANCE were used as indicators of financial capacity and liquidity. The correlation 
analysis reveals relationships between spending intensity, credit utilization, and repayment behavior, which 
serve as the foundation for clustering financial archetypes.
